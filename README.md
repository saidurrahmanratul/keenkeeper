# KeenKeeper

> A personal relationship manager that helps you stay meaningfully connected with the people who matter most.

---

## 🛠 Technologies

- **React 19** — UI framework
- **Vite** — build tool & dev server
- **Tailwind CSS** — utility-first styling
- **React Router DOM** — client-side routing
- **Recharts** — data visualization
- **Lucide React** — icon library

---

## Key Features

- **Friend Dashboard** — View all your friends at a glance with relationship status indicators (on-track, needs attention) and contact tags, so you always know who to reach out to next.

- **Interaction Timeline** — Log and browse every interaction — calls, texts, video chats, and meetups — with filterable history to keep a clear record of your connections.

- **Friendship Analytics** — Visualize your communication habits with a breakdown of interaction types, helping you identify patterns and maintain balanced relationships.

---

## Getting Started

```bash
npm install
npm run dev
```
